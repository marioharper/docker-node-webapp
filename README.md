# docker-node-webapp

## Install
```bash
  $ git clone https://github.com/marioharper/docker-node-webapp.git
  $ docker build -t node-webapp ./docker-node-webapp
```

## Run
```
  $ docker run -p 8080:8080 -d node-webapp
```
